# 📝 Minimalist To-Do List Application

A beautiful, lightweight, and modern desktop To-Do List application built entirely in Python using standard **Tkinter**. 

Unlike standard, dated Tkinter interfaces that resemble 90s operating systems, this app utilizes custom color palettes, simulated padding, and flat UI elements to deliver a sleek, minimalist aesthetic natively on macOS, Windows, and Linux.

## ✨ Features

* **Modern Flat UI:** Designed using custom fonts and hex color palettes with a clean, borderless layout.
* **Pure Python Integration:** Zero external dependencies—runs flawlessly using standard libraries.
* **Dynamic Scrollable Task Pane:** Seamlessly handles long list overflows using an underlying Tkinter Canvas system.
* **Responsive Row Architecture:** Each newly generated task is housed within its own cleanly padded visual card container.
* **macOS Optimization:** Configured to dynamically bring the interface window to focus on execution launch.

## 🚀 Getting Started

You only need **Python 3.x** installed on your machine. Tkinter comes bundled with standard Python installations.

> **Mac Users Note:** If Tkinter is missing on your specific environment, you can install it using Homebrew via terminal: `brew install python-tk`
